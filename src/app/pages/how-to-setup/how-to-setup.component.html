Accounts

You will need a Firebase & Open Weather account and API Keys

Firebase

Signup for a firebase account and project using the Firebase Console. Make sure you setup Authentication and allow users to signin using email & password. Make sure you setup Firestore to store what cities you follow

Setup a webapp and copy your config into environment.development.ts

Open Weather

Signup for an OpenWeather account at https://openweathermap.org.

Get a Weather API Key and then add it to appID within environment.development.ts
